This is a repository dedicated to uBlock filters and related lists that I have made to fill in missing gaps - lists that, to my knowledge, do not exist elsewhere.

#### Hulu Spoilers
The first list, which is the one that prompted me to start this repository, is dedicated to avoiding episode spoilers on Hulu. When I was watching a series with a friend, he would visit the series page to go to each episode, and this page contains many thumbnails and descriptions for each episode. For certain series, these can *spoil* information that was supposed to be delivered later. To avoid this in the future, thumbnails and descriptions have to go.

Currently, the Hulu Spoilers filter list can be obtained directly from this URL: https://raw.githubusercontent.com/Dgrif12/uBlock-Filters/refs/heads/main/Hulu-Spoilers

#### edX Certificate Promotions and Popups on Course Audits
The second list is the first attempt to hide the overlay that appears on every other page of a course. Additionally, it hides the course sidebar, which I have to admit that I do not know if it serves a purpose besides selling the certificates. It has been tested on only one course, so consider it a rough draft.

The edX filters can be accessed via: https://raw.githubusercontent.com/Dgrif12/uBlock-Filters/refs/heads/main/edX-Audit-Certificate
